## 12단계 - React에서 State 시작하기 - Hook으로 State 사용하기

- 각 자바스크립트 파일은 모듈
- 각 컴포넌트는 각자의 모듈에 있는 것이 이상적
- 컴포넌트에는 State가 있음
    - 특정 컴포넌트에 대한 데이터나 정보
- 원래는 함수 컴포넌트에는 Steate가 없엇음
    - 그러나 Hook 등장 이후에 함수 컴포넌트도 State를 가질 수 있게 됨
